Koncerta biļetes

Projekta apraksts

Šis projekts ir interaktīva mājaslapa koncerta biļešu cenas aprēķinam. 
Lietotājs var ievadīt biļešu skaitu, izvēlēties biļešu tipu un piemērot studenta atlaidi. 
Cena tiek aprēķināta dinamiski, izmantojot JavaScript, bez lapas pārlādes.


Funkcionalitāte

- Lietotāja ievade (biļešu skaits, biļešu veids, atlaide)
- Datu validācija (nepieciešams pozitīvs skaitlis)
- Cenas aprēķins, izmantojot objektu ar cenām
- Studenta atlaide (-20%)
- Rezultāta dinamiska attēlošana HTML dokumentā
- Paziņojuma krāsas maiņa atkarībā no cenas
- CSS noformējums (dark mode, neon stils)

Izmantotās tehnoloģijas

- HTML
- CSS (glassmorphism, gradienti, animācijas)
- JavaScript (DOM manipulācija, notikumu apstrāde)


Lietošanas instrukcija

1. Atver index.html pārlūkprogrammā.
2. Ievadi biļešu skaitu.
3. Izvēlies biļetes tipu.
4. Ja nepieciešams, atzīmē studenta atlaidi.
5. Nospied pogu "Aprēķināt".
6. Rezultāts tiks parādīts zem pogas.
